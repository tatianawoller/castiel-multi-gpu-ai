# Data

Data sets required for this training.


## What is it?

1. {download}`pandas_data_analysis_questions_train.jsonl`: JSONL file containing a dataset
   of questions and answers about the pandas Python library.  This dataset will
   be used for training.
1. {download}`pandas_data_analysis_questions_test.jsonl`: JSONL file containing a dataset
   of questions and answers about the pandas Python library.  This dataset will
   be used for validation.
